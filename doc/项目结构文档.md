# 项目结构文档

## 1. 技术栈

| 层级 | 技术 | 版本 | 说明 |
|---|---|---|---|
| 前端框架 | Vue.js | 3.5.x | Composition API + `<script setup>` |
| 构建工具 | Vite | 6.x | 开发端口 3493 |
| UI / 样式 | Tailwind CSS | 3.4.x | 暗色主题，CSS 变量驱动 |
| 状态管理 | Pinia | 3.x | Composition Store 风格 |
| 路由 | Vue Router | 4.x | History 模式 |
| HTTP 客户端 | Axios | 1.7.x | 统一拦截、错误处理 |
| 图标 | Font Awesome | 6.x | vue-fontawesome 3.x |
| 桌面壳 | Electron | 33.x | electron-vite 2.x 构建 |
| 后端框架 | FastAPI | 0.115+ | 异步，Pydantic 校验 |
| 运行时 | Uvicorn | 0.34+ | 端口 3492 |
| 配置管理 | pydantic-settings | 2.7+ | `.env` + 环境变量 |
| 外部 API | qqmusic-api-python | 0.5+ | QQ 音乐数据获取 |
| HTTP 客户端 | httpx | 0.28+ | 异步重定向解析、文件大小探测 |
| Python | Python | 3.12+ | uv 管理依赖 |

## 3. 文件功能说明

### 3.1 后端 (`backend/`)

| 文件 | 功能 |
|---|---|
| `app/main.py` | FastAPI 应用入口，注册 CORS 中间件、路由前缀 `/api/v1`、健康检查 `/health` |
| `app/api/song.py` | 歌曲路由层，三个 POST 端点：`/song/search`、`/song/albumImg`、`/song/songUrl` |
| `app/services/song.py` | 核心业务逻辑：搜索单曲/歌单、获取专辑封面 URL、获取歌曲下载链接、解析 302 重定向 |
| `app/schemas/song.py` | 歌曲相关 Pydantic 模型：请求体 (`SearchRequest` 等)、响应体 (`SongItemResponse` 等) |
| `app/schemas/common.py` | 通用模型：`ApiResponse` 统一响应结构、`PaginatedResponse` 分页、`ErrorCode` 错误码枚举 |
| `app/schemas/response.py` | 响应构造 helper：`success()` / `error()` 快捷函数 |
| `app/core/config.py` | pydantic-settings 配置类，从 `.env` 读取 host / port / CORS / 日志参数 |
| `app/credential/get_credential.py` | 从 `credential.json` 加载 QQ 音乐登录凭证 |
| `app/credential/request_credential.py` | 二维码登录脚本，获取凭证并写入 `credential.json` |
| `app/credential/credential.json` | QQ 音乐 API 凭证数据 (musicid / credential 等) |
| `app/qqmusic/client.py` | `qqmusic_api.Client` 全局单例管理 |
| `app/utils/exception.py` | `ServiceException` 统一业务异常类 |
| `app/utils/logging.py` | 日志配置，RotatingFileHandler + 控制台输出 |
| `pyproject.toml` | uv 项目元数据与依赖声明 |

### 3.2 前端 (`sys_vue/`)

| 文件 | 功能 |
|---|---|
| `src/main.js` | Vue 应用初始化，挂载 Router / Pinia / Tailwind / Font Awesome 图标库 |
| `src/App.vue` | 根组件 |
| `src/router/index.js` | 路由配置：`/` → `/home` 重定向，`/localmusic` 本地音乐页 |
| `src/views/Index.vue` | 主容器页面，侧边栏导航 + `<router-view>` 子路由出口 |
| `src/views/Home.vue` | 首页视图：链接搜索、歌曲列表展示、播放控制 |
| `src/views/LocalMusicGet.vue` | 本地音乐扫描页面 |
| `src/stores/Song.js` | Pinia Store：播放列表、当前歌曲、播放状态、播放模式 (顺序/随机)、上下曲切换 |
| `src/api/song.js` | 歌曲相关 API：`searchSongs` / `getAlbumImages` / `getSongUrls` |
| `src/api/index.js` | API 模块统一导出 |
| `src/utils/request.js` | Axios 实例封装：请求拦截 (Token 注入)、响应拦截 (统一错误处理) |
| `src/utils/toast.js` | 自定义 Toast 提示，替代 Element Plus ElMessage |
| `src/components/MusicPlayer.vue` | 底部固定播放器：播放控制、进度拖拽、音量调节、播放模式切换 |
| `src/components/TitleBar.vue` | Electron 自定义标题栏：最小化 / 最大化 / 关闭 |
| `src/components/ErrorDisplay.vue` | 错误状态展示，支持重试按钮 |
| `src/components/LoadingSpinner.vue` | 加载状态展示，支持自定义文本和插槽 |
| `src/components/base/BaseButton.vue` | 统一按钮组件 (primary / outline / ghost / danger) |
| `src/components/base/BaseInput.vue` | 暗色输入框，支持前缀/后缀插槽、清除按钮 |
| `src/components/base/BaseSlider.vue` | 自定义 range 滑块，渐变轨道 + 发光 thumb |
| `src/components/base/BaseSelect.vue` | 暗色下拉框，支持 v-model、自定义显示字段 |
| `src/components/base/BasePagination.vue` | 完整分页器：页码、上/下一页、每页条数、总条数 |
| `src/components/base/BaseProgressBar.vue` | 渐变进度条，支持百分比和动画 |
| `src/components/business/SongListTable.vue` | 歌曲列表表格，columns 配置驱动，支持选中、封面、操作列 |
| `vite.config.js` | Vite 构建配置：别名 `@`、端口 3493 |
| `tailwind.config.js` | Tailwind 主题配置（暗色主题、CSS 变量） |
| `.env.development` | 开发环境变量 (VITE_API_BASE_URL) |
| `.env.production` | 生产环境变量 |

### 3.3 Electron (`sys_electron/`)

| 文件 | 功能 |
|---|---|
| `main.js` | 主进程：创建无边框窗口、管理 Flask 子进程（生产模式）、注册 IPC 处理器（目录选择/文件下载/窗口控制）、应用菜单 |
| `preload.js` | 预加载脚本：通过 `contextBridge` 暴露 `electronAPI`（selectDirectory / selectFiles / downloadFile / 窗口控制等） |
| `menu.js` | 应用菜单配置（未使用，菜单已在 main.js 中内联定义） |
| `electron.vite.config.js` | electron-vite 构建配置：main / preload / renderer 三端构建，共享 sys_vue 的 Tailwind 配置 |
| `package.json` | Electron 依赖声明、打包配置 (electron-builder，Linux AppImage / Windows NSIS) |
